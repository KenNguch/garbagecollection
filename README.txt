--------INSTALLATION--------
After extracting the zip file in the root directory of your local server/online server
Go to the folder named INSTALL
Find a file named truckbooking.sql
Go to phpmyadmin and create a database named truckbooking
Import the file we named above truckbooking.sql to create all the required tables for our system
After successfull import you can now use the system
